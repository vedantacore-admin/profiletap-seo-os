## 2. Distribution Workflow by Content Type

### Hub / Category Pages (Commercial)

1. **Day 0 (Publish):** Submit URL to Search Console for indexing
2. **Day 0:** Share on LinkedIn with business-angle positioning
3. **Day 1:** Share on Twitter/X with key benefit hook
4. **Day 2:** Create Instagram carousel highlighting key features
5. **Day 3-7:** Share in 2-3 relevant LinkedIn/Facebook groups
6. **Week 2:** Repurpose key sections as Quora answers (link back to full page)
7. **Ongoing:** Add to email signature, newsletter if applicable

### Blog Posts (Educational)

1. **Day 0:** Submit URL to Search Console
2. **Day 0:** Share on LinkedIn with insight/takeaway hook
3. **Day 0:** Share on Twitter/X with thread summarizing key points
4. **Day 1:** Post on relevant Reddit threads (only if genuinely helpful, not promotional)
5. **Day 2-3:** Repurpose as Quora answers for matching questions
6. **Week 2:** Share in relevant community groups
7. **Month 1:** Syndicate to Medium (set canonical to original URL)

### Comparison Pages

1. **Day 0:** Submit to Search Console
2. **Day 0:** LinkedIn post — "Switching from {competitor}? Here's what to know"
3. **Day 1:** Twitter thread comparing features
4. **Day 3-7:** Answer relevant Quora questions about the competitor
5. **Ongoing:** Monitor competitor mentions on social — engage helpfully

## 3. Social Media Posting Guidelines

### LinkedIn

- **Format:** Short text (150-300 words) + link or carousel
- **Hook:** Lead with a bold statement, question, or stat
- **CTA:** "Full breakdown on our site" (don't say "link in comments")
- **Hashtags:** 3-5 relevant (#DigitalBusinessCard #NFC #IndiaStartup)
- **Timing:** Tue-Thu, 9-11 AM IST

### Twitter/X

- **Format:** Thread (5-8 tweets) OR single tweet with link
- **Hook:** First tweet must stop the scroll — stat, question, or contrarian take
- **CTA:** Last tweet links to the page
- **Hashtags:** 1-2 max
- **Timing:** Mon-Fri, 10 AM - 1 PM IST

### Instagram

- **Format:** Carousel (5-7 slides), Reels (30-60 sec), Stories
- **Style:** Clean, branded, text-heavy (Instagram users read carousels)
- **CTA:** "Link in bio" pointing to the relevant page
- **Hashtags:** 10-15 in first comment
- **Timing:** Mon/Wed/Fri, 12-2 PM IST

## 4. Community Engagement Rules

### Do

- Provide genuine value before linking to your content
- Answer the question fully, then add "I wrote more about this here: [link]"
- Match the community tone (don't sound corporate in casual communities)
- Engage with others' posts before sharing your own
- Disclose your affiliation when relevant

### Don't

- Drop links without context
- Post the same content in multiple groups simultaneously
- Use fake accounts or sock puppets
- Spam the same link repeatedly
- Ignore community rules or moderator guidelines
- Self-promote in communities that explicitly prohibit it

## 5. Indexing Acceleration

### For New Pages

1. Submit URL via Google Search Console URL Inspection tool
2. Ensure the page is linked from at least 3 internal pages
3. Include the page in the XML sitemap
4. Share on social media (social signals help discovery)
5. If not indexed within 7 days, re-check for technical issues

### For Updated Pages

1. Update `<lastmod>` in sitemap
2. Submit URL via Search Console URL Inspection
3. Share the updated version on social (creates new discovery signals)
4. If critical update, ping Google via Search Console API

### Monitoring

- Check new pages in Search Console Coverage report within 7 days
- Flag any page that isn't indexed within 14 days for investigation

## 6. Email Distribution (When Applicable)

### For Existing Users

- Include 1-2 new SEO pages per email with brief description
