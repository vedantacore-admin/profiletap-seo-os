## Mandatory Page Elements By Type

### Homepage

- clear smart identity platform positioning
- use-case overview
- product capability blocks
- primary conversion CTA

### Category page

- definition of the category
- benefits and use cases
- product walkthrough
- CTA to create profile or request demo

### Use-case page

- audience-specific pain points
- segment-specific use cases
- tailored feature framing
- segment-relevant CTA

### Comparison page

- competitor alternative framing
- comparison table
- reasons to switch
- FAQ covering migration or switching concerns

### Blog page

- direct answer to the query
- educational depth
- internal links to commercial pages
- CTA block that routes readers to the correct product page
